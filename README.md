# タクシー売上ノート

タクシー乗務員が売上をスマホだけで記録し、Excel / CSV / JSONバックアップに出力できる静的PWAです。データはIndexedDBに保存され、サーバー送信や外部集約は行いません。

## 起動方法

macOSでは `▶ 起動 売上ノート.command` を開くと、プロジェクトルートで `python3 -m http.server 8765` を起動し、`http://127.0.0.1:8765` をブラウザで開きます。

手動で起動する場合:

```bash
cd /Users/suzukitakao/Desktop/タクシー売上ノート
python3 -m http.server 8765
```

`index.html` を直接開いても基本機能は動きますが、Service Worker / PWAのオフラインキャッシュは `http://127.0.0.1` またはHTTPS環境で有効になります。

## 使い方

- 初回起動時に乗務員名を入力し、「ID/PWを設定」または「ロックなし」を選択します。
- 「入力」から日付、営業回数、走行km、売上内訳、勤務時刻、経費、メモを保存します。
- 売上合計は `現金 + カード + IC + QR + チケット` で自動計算します。
- ホームと履歴では月別の合計と一覧を確認できます。
- 出力画面でExcel(.xlsx)、CSV(UTF-8 BOM付き)、JSONバックアップをダウンロードできます。
- 設定画面で乗務員名、ロック方式、バックアップ復元、全データ削除を操作できます。

## 配布

ビルド工程はありません。GitHub Pagesなどの静的ホスティングにこのリポジトリを置けばPWAとして配布できます。スマホでは公開URLを開き、ブラウザの「ホーム画面に追加」からアプリ化できます。

## データとロックの制約

データは端末内のIndexedDBだけに保存されます。端末紛失、ブラウザデータ削除、機種変更に備えて、定期的にJSONバックアップを書き出してください。

ID/PWロックは画面をすぐに見られないようにするためのソフトな鍵です。IndexedDB自体は暗号化していないため、端末やブラウザ開発者ツールにアクセスできる人から完全に隠すものではありません。

## テスト

`http://127.0.0.1:8765/_selftest.html` を開くと、ブラウザ内で計算、CSV、Excel行マッピング、認証、IndexedDB CRUDのアサーションを実行します。画面上部が `PASS` になれば成功です。
